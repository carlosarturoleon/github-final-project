# Simple Interest Calculator

## Project Description

This project is a simple interest calculator implemented as a Bash shell script. It allows users to compute simple interest based on three user-provided inputs:

- **Principal** – the initial amount of money
- **Rate of Interest** – the annual interest rate (in percentage)
- **Time Period** – the duration (in years)

## Formula

```
Simple Interest = (Principal × Rate × Time) / 100
```

## Usage

Run the script and follow the prompts:

```bash
bash simple-interest.sh
```
